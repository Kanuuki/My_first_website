Red hat, Blue Hat

I've always wanted to learn how to hack like those guys in those corny movies. Now that my brain is developed I've realized those are absolutley fake and hilarious how bad they are.

When I first started learning about hacking, I was directed to a website. It's name is OverTheWire and It teaches you how to navigate the terminal and find what are called "flags" that unlock the path to the next level. Like all games, it started out eazy. I got stuck on level 14 because of how confusing it is. A good thing about that is I learned how to use the CLI and am now more comfortable using it than a file system GUI

After I did some of Bandit in OverTheWire, I went on to a website called tryhackme. It has a decent amount of free content that it offers, and for only $10 you unlock a bunch of features and courses that teach you about penetration testing and the many ways of getting inside of something securely locked down. The website also teaches about cyber defense and how to protect yourself from attacks and malware. Since I've never done any of this before in my life, I started out with Complete Beginner and even that is whooping my ass. There's so much to learn about and understand and it's overwhelming
